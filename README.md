Для того чтобы редачить UI нужно экспортировать файл который будет править софт, для этого

1. Заходим в игру
2. Открываем HUD Редактор
3. Слева внизу жмём кнопку Экспорта
4. Обзываем конфиг и жмём экспорт
5. Открываем его в TL HUD EDITOR
6. Редактируем как хотим
7. В игре, в HUD Редакторе нажимаем кнопку Изменить
8. Справа от названия файла будет стрелка
9. Жмём загрузить
10. Кайфуем
